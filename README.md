## 它能做什么

说实话，光说"缩写替换"有点低估它了，来列一下真正好用的功能：

**输入增强**
- 设置缩写，触发后自动展开为富文本、纯文本或代码片段
- 支持光标占位——展开后光标停在你需要填写的位置，直接填就行
- 支持插入当前日期、剪贴板内容、自定义变量等动态内容

**脚本执行**（付费版专属）
- 片段里可以内嵌 JavaScript、Python、Bash Shell、AppleScript 代码
- 触发缩写时实时运行脚本，返回处理结果
- 这个功能有点厉害，基本等于把一部分自动化工作也接管了

**自定义模板**
- 触发后弹出填写面板，支持单行/多行输入、下拉选择、复选框
- 填完一按确认，整段内容连同你填的部分一起输出

**跨平台云同步**
- 支持 Windows 10+ 和 macOS 13+ 双平台
- 可通过 GitHub Gist、Gitee 或 WebDAV 同步数据
- 公司 Mac 设的片段，回家 Windows 上一样用

**剪贴板历史**（付费版）
- 保存最近 100 条剪贴板内容
- 支持全文搜索、收藏，点击直接粘贴

**截图 & 标注**（新增）
- 区域截图、窗口截图、全屏截图
- 支持 OCR 文字识别，可以框选截图里的代码直接复制
